Bonjour !

Bienvenue dans le fichier d'aide de votre application jMusicHub : vous trouverez ici des éléments nécessaires à son bon fonctionnement.

1. Vous pouvez compiler et exécuter votre application directement depuis un terminal en vous plaçant tout simplement dans le dossier jMusicHub.
La commande qui vous aidera à compiler et à run l'application est : bash compile.sh && bash run.sh

2. Une documentation a été implémentée tout au long du code pour vous aider à mieux comprendre les choix de classes, d'interface et de méthodes.

3. Pour générer la Javadoc, il faut compiler et exécuter javadoc.sh qui se trouve à la racine du dossier. Il n'a pas pu être intégré au script de base de l'application suite à un conflit entre JDK et Windows, nous avons donc trouvé une solution de contournement.
Il faudra ensuite aller dans le dossier doc et ouvrir le fichier index.html avec un navigateur. 

5. Vous pouvez contacter le service client à : madad@et.esiea.fr ou louaty@et.esiea.fr pour toute question supplémentaire.
